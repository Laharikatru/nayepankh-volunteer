# NayePankh Volunteer App 🕊️

A mobile app built with React Native for NayePankh Foundation — helping students discover internships and register as volunteers.

## Screens

1. **Splash Screen** — Animated logo with NayePankh branding
2. **Home Screen** — Impact stats, quick navigation, mission statement
3. **About Screen** — Foundation story, values, team info
4. **Internships Screen** — 8 internship roles with filtering by category
5. **Register Screen** — Full volunteer registration form with validation
6. **Success Screen** — Confirmation with application details and next steps

## Features

- ✅ Animated splash screen
- ✅ Impact statistics (10,000+ students, 500+ volunteers)
- ✅ 8 internship categories with expandable cards
- ✅ Category filtering (Tech, AI/ML, Design, Data)
- ✅ Volunteer registration form with validation
- ✅ Success confirmation screen
- ✅ NayePankh brand colors (deep blue + orange)
- ✅ Clean, premium mobile UI

## Tech Stack

- React Native
- Expo SDK 51
- JavaScript
- React Native Animated API

## Setup

```bash
npm install
npx expo start
```

Scan QR code with Expo Go app on your phone.
